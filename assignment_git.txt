2. Here are a few git and GitHub commands we usually use in software development, please explain the meanings and use cases of them.

  ● git status
	查看目前資料庫的情形：資料的狀態是在Modified、Staged還是Commited。
 
  ● git add
	將資料從自己的資料夾（Modified)移動到暫存區（Staged)。

  ● git commit
	將資料從暫存區（Modified)移動到版本庫（Commited)裡。

  ● git log
	查看自己資料各個版本。


  ● git push [ Repo_name ] [ Branch_name ]


  ● git remote -v


  ● git branch


  ● fork
